RF Walkie-Talkie + FM Radio System

Custom PCB RF communication system integrating transceiver and FM receiver functionality.

RF PCB Embedded Systems Audio

Overview

A custom PCB-based RF communication build combining walkie-talkie functionality with FM radio reception. The goal was to integrate multiple subsystems into one physical board and validate real transmit/receive behavior.

What I Built

  • SA818 transceiver integration
  • SI4730 FM receiver integration
  • OLED display and user controls
  • Audio output circuitry
  • Power regulation section
  • Custom PCB assembly and debugging

Engineering Challenges

  • RF signal behavior and antenna effects
  • Clock oscillator instability
  • Grounding and PCB layout considerations
  • Battery current limitations
  • System integration across multiple modules

Results

Successfully transmitted and received voice signals and demonstrated FM radio functionality. Debugging involved measurements, component testing, and iterative hardware refinement.

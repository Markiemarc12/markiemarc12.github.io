RF Walkie-Talkie + FM Radio System

Custom PCB RF communication system integrating transceiver and FM receiver functionality.

RF PCB Embedded Systems Audio Debugging

What is this?

This project was a custom PCB-based RF communication build combining walkie-talkie functionality with FM radio reception. The goal was to integrate multiple subsystems into one physical board and validate real transmit/receive behavior.

What I Built

  • Custom PCB assembly
  • SA818 transceiver integration
  • SI4730 FM receiver integration
  • OLED display and user controls
  • Audio output circuitry
  • Power regulation section

Technical Focus

  • RF module integration
  • Clock and oscillator behavior
  • Audio path debugging
  • Power delivery
  • PCB layout and grounding

Engineering Challenges

  • RF signal behavior and antenna effects
  • Clock oscillator instability
  • Grounding and PCB layout considerations
  • Battery current limitations
  • System integration across multiple modules

Results / Current Status

Successfully transmitted and received voice signals and demonstrated FM radio functionality. Debugging involved measurement, component testing, and iterative hardware refinement.

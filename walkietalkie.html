RF Walkie-Talkie + FM Radio System

Custom PCB RF communication project integrating transceiver and FM receiver functionality.

← Back to Home

Overview

Designed and built a custom PCB-based RF communication system integrating walkie-talkie functionality, FM radio reception, display integration, audio output, and power regulation.

Technical Focus

  • SA818 transceiver module
  • SI4730 FM receiver
  • OLED display interface
  • Audio output circuitry
  • Power regulation and battery constraints
  • Custom PCB assembly and debugging

Engineering Challenges

  • RF debugging and signal behavior
  • Clock oscillator issues
  • Grounding and PCB layout considerations
  • Battery current limitations
  • System integration across multiple modules

Results / Current Status

Successfully transmitted and received voice signals while demonstrating FM radio functionality. Debugging involved measurements, component testing, and iterative hardware refinement.
